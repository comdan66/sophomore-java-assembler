# Welcome To OA's 大二作品 Java Assembler!
為系統程式課程實作 Assembler 組譯器，主功能是將 SIC XE Literal 的組合語言編譯成機械語言，借由實作這個轉換器的同時，更加的了解組合語言與機械語言間的關係與原理。當時會選擇使用 Java 是因為 GUI 介面更容易表現出理想的操作介面，更可以讓使用者更加方便轉換組合語言。
